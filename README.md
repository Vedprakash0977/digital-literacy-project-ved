# 📚 Digital Literacy Project

This repository contains all the tasks completed as part of the **Digital Literacy Course** at VIT Bhopal. The project focuses on improving awareness about digital tools, online safety, and professional communication skills.

---

## 🚀 Project Overview

The goal of this project is to develop essential digital skills required for students, including:
- Digital awareness
- Online safety practices
- Coding platforms exposure
- Professional communication

---

## 📂 Project Structure

### 🔹 Task 1: Digital Literacy Awareness Infographic
- Created an infographic explaining digital literacy
- Covers online safety, digital tools, and responsible usage

---

### 🔹 Task 2: Student Digital Portfolio
- Created professional profiles on platforms like:
  - LinkedIn
  - GitHub
- Added bio, education, and skills
- Built a foundation for future career opportunities

---

### 🔹 Task 3: Coding & Collaboration Platforms
- Practiced coding on platforms like:
  - HackerRank / CodeChef
- Explored collaboration using Google Workspace tools
- Created and shared a Google Form for awareness

---

### 🔹 Task 4: Professional Email & Etiquette
- Wrote formal emails:
  - To professor (extension request)
  - To company (internship request)
- Learned proper email structure and etiquette

---

### 🔹 Task 5: Cybercrime Awareness Case Study
- Studied real-life cybercrime cases:
  - Phishing
  - Online fraud
- Created prevention checklist and awareness guide

---

## 🛠️ Tools & Technologies Used

- Canva (for infographic design)
- Google Forms & Docs
- GitHub (for project hosting)
- Coding platforms (HackerRank, CodeChef)

---

## 📈 Learning Outcomes

- Improved understanding of digital literacy
- Learned safe and responsible internet usage
- Built professional online presence
- Enhanced communication skills
- Gained exposure to coding platforms

---

## 🙌 Conclusion

This project helped in developing essential digital and professional skills required in today’s world. It serves as a foundation for future learning and career growth.
